# 🏥 MediFolio

A secure Medical Record Management System built with **Flask** and **Firebase Firestore** to help healthcare providers manage patient records efficiently and securely.

---

## 📌 Overview

MediFolio is a web-based application that simplifies the storage, retrieval, and management of patient medical records. It provides a secure platform for healthcare professionals to maintain patient information with an intuitive user interface.

---

## ✨ Features

- 🔐 Secure Admin Login
- 👤 Add and Manage Patient Records
- 🔍 Search Patients by Name, ID, Blood Group, and Fingerprint
- 📄 Generate Medical Reports in PDF
- ☁️ Firebase Firestore Database Integration
- 📱 Responsive User Interface
- 🛡️ Secure Backend using Flask

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/Anshul-Taywade/Medifolio.git
```

### Navigate to the Project

```bash
cd Medifolio
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python main.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

## 🔒 Security

The following files are intentionally excluded from the repository:

- Firebase Admin SDK Key
- Environment Variables (.env)
- Uploaded Patient Files
- Virtual Environment (venv)
